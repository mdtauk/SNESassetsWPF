# S-CD-CAD Asset Viewer

This ia a WPF app designed by myself, and code mostly generated with the assistance of CoPilot, for the viewing of files originally created using the S-CG-CAD software for Super Nintendo game development.

Using information gathered by RetroReversing on their very useful website, and information by the dedicated community who are researching the files created by the A Link to the Past designers and developers during the creation of the game.
https://www.retroreversing.com/snes-file-formats

Inspired by apps like CGX Viewer and HyperCG-CAD

This is a project to help me understand the file formats, and help in the investigation of content that is being explored by amazing people such as GlitterBerri and the people at The Cutting Room Floor.
https://tcrf.net/Development:The_Legend_of_Zelda:_A_Link_to_the_Past

Its not intended for general use, and may or may not ever be relased, but I created a repo to help manage any roll backs if things break and to allow others to look at the code, if I ever seek out advice or assistance while working on this side project.
